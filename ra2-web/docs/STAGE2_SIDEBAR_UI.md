# 红警2 Web版 - 阶段2开发进度

## 📋 侧边栏UI系统 ✅ 已完成

### 实现的组件

#### 1. ResourceDisplay.tsx - 资源显示
- **位置**: 屏幕右上角（侧边栏左侧）
- **功能**:
  - 显示资金 ($XXXXX)
  - 显示电力状态 (消耗/产出)
  - 显示单位数量
- **特性**:
  - 电力不足时变橙色警告
  - 半透明黑色背景
  - 仿红警2金色数字风格

#### 2. RadarMap.tsx - 雷达小地图
- **位置**: 侧边栏顶部
- **功能**:
  - 实时显示地图地形
  - 显示建筑和单位位置
  - 战争迷雾效果（黑色/灰色）
  - 相机视口框
  - **点击跳转**: 点击雷达任意位置移动相机
- **颜色编码**:
  - 绿色: 玩家单位
  - 蓝色: 玩家建筑
  - 红色: 敌方建筑/单位
  - 黑色: 未探索区域
  - 灰色: 已探索但当前不可见

#### 3. BuildMenu.tsx - 建造菜单
- **位置**: 侧边栏主体区域
- **功能**:
  - 4个标签页: 建筑/防御/步兵/车辆
  - **建筑标签页**: 电厂、矿厂、兵营、战车工厂等
  - **防御标签页**: 光棱塔/特斯拉线圈、防空设施
  - **步兵标签页**: 基础步兵、工程师、间谍等
  - **车辆标签页**: 坦克、采矿车、防空车等
- **特性**:
  - 根据阵营显示不同建筑（盟军/苏联）
  - 资金不足时按钮变灰禁用
  - 显示成本和建造图标
  - **生产队列显示**: 选中兵营/战车工厂时显示队列
  - **底部按钮**: 路径点/修复/出售

#### 4. SelectionPanel.tsx - 选中信息面板
- **位置**: 屏幕左下角
- **功能**:
  - **单选单位**: 显示详细信息
    - 单位图标和名称
    - 生命值条（绿/黄/红）
    - 当前状态
    - 攻击力/护甲/速度/视野
    - 老兵等级（如果有）
  - **多选单位**: 按类型分组显示数量
  - **选中建筑**: 显示建筑信息
    - 建筑图标和名称
    - 完整性条
    - 建造进度（如果正在建造）
    - 电力消耗/产出
    - 生产设施标识

#### 5. GameCanvas.tsx - 整合更新
- **布局调整**:
  - 3D画布留出160px右侧空间给侧边栏
  - 顶部状态栏也留出侧边栏空间
- **新增功能**:
  - 游戏时间显示
  - 实时状态更新（每10帧）
  - 雷达点击跳转相机
  - 建筑放置回调
  - 单位生产回调
  - 建筑修复/出售功能

### 引擎更新 (Engine.ts)

#### 新增方法:
1. `setGameManager()` - 支持外部传入游戏管理器
2. `moveCamera(x, y, z)` - 移动相机到指定位置
3. `startBuildingPlacement()` - 开始建筑放置模式
4. `startWaypointPlacement()` - 开始路径点设置
