### 6. 大MIX文件彻底修复 (第二次优化)
**问题**: 第一次优化后仍报 `RangeError: Array buffer allocation failed`

**根本原因**: 第一次优化仍然使用 `new Uint8Array(arrayBuffer)` 保存完整文件内容

**最终解决方案**:
- 使用 `File.slice()` 只读取文件头（通常 < 100KB）
- 保存 `File` 对象引用而非 `Uint8Array`
- 按需读取：需要时再 `file.slice(start, end)` 提取特定文件
- 预加载限制：只加载 <100KB 的SHP，最多50个

**代码对比**:
```typescript
// 旧方式 ❌
const data = new Uint8Array(await file.arrayBuffer()) // 加载整个文件

// 新方式 ✅
const header = await file.slice(0, 100).arrayBuffer() // 只读头部
// 保存 file 引用，需要时再 slice
```

### 7. 测试资源加载卡住修复
**问题**: 点击"使用测试资源"一直卡在"初始化中"

**根本原因**: fetch 请求没有超时，网络问题导致无限等待

**解决方案**:
- 添加 5 秒超时：`AbortController` + `setTimeout`
- 允许 0 资源启动：即使全部失败也能进入游戏
- 详细日志：控制台显示每个文件的加载状态
