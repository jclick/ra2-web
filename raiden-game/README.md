# 雷电 - Raiden Arcade Replica

基于 TypeScript 开发的雷电街机版完整复刻。

## 原版规格

- **开发商**: SEIBU KAIHATSU (西武开発)
- **发行年份**: 1990
- **原始分辨率**: 224×256 (竖屏)
- **颜色数**: 2048色
- **刷新率**: 60Hz

## 特性

### 完全还原的武器系统

**主武器 (红/蓝切换)**
- **Vulcan (红/散弹)**: 8级火力，扩散角度随等级增加
- **Laser (蓝/激光)**: 8级火力，光束数量和宽度随等级增加

**副武器 (M/H切换)**
- **Missile (M/导弹)**: 直线高速火箭
- **Homing (H/追踪)**: 自动追踪最近敌人

### 完整道具系统

| 道具 | 效果 |
|:---|:---|
| 🔴 **R** | Vulcan主武器 |
| 🔵 **L** | Laser主武器 |
| 🟢 **M** | Missile副武器 |
| 🟠 **H** | Homing副武器 |
| ⭐ **P** | 当前武器火力提升 |
| 💣 **B** | 炸弹 +1 |
| ❤️ **1UP** | 额外生命 |
| 🏅 **勋章** | 分数，连续收集价值递增 |
| 🐉 **Miclus** | 西武吉祥物，3000分 |
| 🧚 **精灵** | 死亡时出现，恢复火力+10000分 |

### 敌机类型

- **红色战机**: 基础敌机，瞄准射击
- **蓝色战机**: 快速移动，双发子弹
- **绿色战机**: 正弦波移动，三向散射
- **棕色轰炸机**: 中型，五向扇形弹幕
- **灰色轰炸机**: 中型，环形弹幕
- **武装直升机**: 大型，连续射击
- **地面坦克**: 向上射击
- **BOSS**: 每关大型BOSS，多阶段

### 操作方式

| 按键 | 功能 |
|:---|:---|
| **WASD / 方向键** | 移动战机 |
| **Z** | 射击 (按住连发) |
| **X** | 使用炸弹 |
| **P** | 暂停游戏 |
| **Enter** | 开始游戏 (标题画面) |

**技巧**: 按住Z键时移动速度降低，便于精确避弹。

## 技术栈

- TypeScript 5.3
- Vite
- HTML5 Canvas API
- 原生 ES Modules

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 生产构建
npm run build

# 预览生产版本
npm run preview
```

## 关于原作

雷电 (Raiden) 是 SEIBU KAIHATSU 于1990年推出的纵向卷轴射击游戏。游戏设定在2090年，地球遭受外星入侵，玩家驾驶雷电超音速攻击机进行反击。

原作特点:
- 极具挑战性的弹幕密度
- 双武器切换系统 (Vulcan/Laser)
- 经典的西武龙吉祥物 (Miclus)
- 高分挑战性的勋章收集系统
